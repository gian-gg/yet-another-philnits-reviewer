# 2022S_FE_AM_34

At which layer of the OSI reference model does a router primarily operate?

a) Data link layer
b) Network layer
c) Transport layer
d) Session layer
?
b) Network layer

### Explanation

**Routers** make forwarding decisions based on **IP addresses**, which live
at the network layer (layer 3) of the OSI model. They examine the destination
IP in each packet and pick the next hop from a routing table.

### Why the Others Won't Fit

**a) Data link layer**:

- Switches and bridges operate here using MAC addresses, not IP.

**c) Transport layer**:

- TCP/UDP endpoints live here; routers don't inspect port/segment state for
  basic forwarding.

**d) Session layer**:

- Handles dialog/session establishment between applications — well above a
  router's concern.
