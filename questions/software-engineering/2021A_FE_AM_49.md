# 2021A_FE_AM_49

In the classical **waterfall** development model, which activity immediately
follows the **requirements definition** phase?

a) Coding
b) Testing
c) External design (system design)
d) Deployment
?
c) External design (system design)

### Explanation

The canonical waterfall phase order is:

```
Requirements → External design → Internal design → Coding → Testing → Deployment → Maintenance
```

Right after requirements are frozen, the team produces the **external
(system) design** describing what the system looks like to its users and
surrounding systems.

### Why the Others Won't Fit

**a) Coding** — happens only after both external and internal design.
**b) Testing** — follows coding.
**d) Deployment** — one of the last phases, after testing.
