# philnits-vault — project_source_of_truth

## 1. Project Overview

**philnits-vault** is a simple review app designed for PhilNITS preparation.

Its purpose is to help users:

- take mock exams easily
- practice problems quickly
- focus only on selected topics

The app is built to feel direct, fast, and distraction-free.

---

## 2. Product Goal

Create a straightforward tool that allows users to review and practice PhilNITS questions with minimal friction.

The emphasis is on usability and efficiency rather than design or complexity.

---

## 3. Core Principles

- simplicity over complexity
- speed over visuals
- utility over aesthetics
- focus over distractions

---

## 4. Core Experience

The app should feel like a clean workspace for studying.

Users should be able to:

- quickly start a practice session
- generate a mock exam
- choose which topics to include or exclude
- answer questions without distractions
- review their performance immediately

Everything should be obvious and require little to no learning curve.

---

## 5. Key Features

- **Practice Mode**
  - answer questions one by one
  - get immediate feedback or review after

- **Mock Exam Mode**
  - simulate an exam session
  - view score at the end

- **Topic Selection**
  - include specific topics
  - exclude weak or irrelevant topics
  - fully control what appears in a session

- **Review Flow**
  - revisit incorrect answers
  - understand mistakes

---

## 6. User Flow

### Practice

1. Open app
2. Select practice
3. Choose topics (include/exclude)
4. Start answering
5. Review results

### Mock Exam

1. Open app
2. Select mock exam
3. Choose topics (optional)
4. Take exam
5. View score and review

---

## 7. Scope

### In Scope

- practice questions
- mock exams
- topic-based filtering
- simple result summaries

### Out of Scope

- complex gamification
- social features
- unnecessary visual design
- anything that slows down the core experience

---

## 8. Product Identity

**philnits-vault** should feel like:

- a personal exam toolkit
- a focused study environment
- a fast way to drill knowledge

It should not feel like a social app or a visually heavy platform.

---

## 9. Success Criteria

The app is successful if a user can:

- start practicing within seconds
- easily control what topics to study
- complete mock exams without friction
- quickly review mistakes

---

## 10. Summary

**philnits-vault** is a minimal, utility-first study tool.

Its goal is simple:
Make PhilNITS preparation fast, focused, and effective.
